# Release Notes - Version 1.1.2 (Version Code 12)

## Release Name

Version 1.1.2 - WorkManager Configuration Fix

---

## Release Notes

### en-GB

Bug fixes and performance improvements:

- Fixed WorkManager initialization configuration issue
- Improved app startup performance
- Enhanced system stability
- Minor bug fixes and optimizations

### tr-TR

Hata düzeltmeleri ve performans iyileştirmeleri:

- WorkManager başlatma yapılandırma sorunu düzeltildi
- Uygulama başlatma performansı iyileştirildi
- Sistem kararlılığı geliştirildi
- Küçük hata düzeltmeleri ve optimizasyonlar

---

## Technical Changes

- Fixed lint error: Removed default WorkManager initializer from AndroidManifest
- Added explicit InitializationProvider removal for on-demand WorkManager configuration
- Updated version code from 11 to 12
- Updated version name from 1.1.1 to 1.1.2
